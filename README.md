# EDA, and Machine Learning on Wine Data

## Introduction
In this repository, we have utilized the wine data and explored it through Exploratory Data Analysis in order to understand how differnet paramters in wine production effect the overall quality of wine, and how these factors are inter-related with each other.

## Analysis Workflow
Data folder contains the required CSV files, and the jupyter notebook `eda_ml_wine.ipynb`. Repository also contains the visual.py which contains the custom functions that has been utilized in the notebook.
